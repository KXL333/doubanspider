目录文件说明：

模块一：load_tag_url.py（独立文件）
---------------------------------------
实现豆瓣图书信息所有标签信息的爬取，并图书的标签信息写入到Redis数据库中，此模块可使用rquests简单实现。

模块二：master（主爬虫项目目录）
---------------------------------------
负责从Redis中获取每个图书标签，并分页式的爬取每本图书的url信息，并将信息写入到redis中。

模块三：slave（丛爬虫项目目录）
----------------------------------------
负责从Redis中获取每个图书的url地址，并爬取对应的图书详情，将每本图书详情信息写回到redis数据库中。

模块四：item_save.py（独立文件）
----------------------------------------
负责从Redis中获取每本图书的详情信息，并将信息依次写入到MySQL数据中，作为最终的爬取信息。

模块五：myweb（项目目录）
----------------------------------------
将爬取的数据信息从mysql数据库中分页输出web网页中。
